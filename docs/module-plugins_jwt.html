Methods

(static) jwtPlugin(fastify, opts)

Verifies if the JWTs that came in the cookies of the request are valid. If the access token is valid, it allows the request. If it is not, but the refresh token is valid, it allows the request and calls the user microservice that generates a new token pair. If neither is valid, it doesn't allow the request.

Parameters:
Name Type Description
fastify The fastify instance.
opts The options for the fastify plugin
